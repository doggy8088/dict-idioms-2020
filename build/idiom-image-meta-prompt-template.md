# 成語圖片 Meta-Prompt Template

```markdown
請根據以下成語資料，生成一張 512x512 的方形圖片。

成語：{{成語}}
核心釋義：{{釋義}}
語義重點：{{用法說明-語義說明}}
使用情境：{{用法說明-使用類別}}
關鍵概念：{{關鍵概念}}
近義參考：{{近義成語}}
反義排除：{{反義成語}}
視覺策略：{{視覺策略}}
主要主體：{{主要主體}}
場景設定：{{場景設定}}
核心視覺關係：{{核心視覺關係}}
構圖方式：{{構圖方式}}
視覺風格：{{視覺風格}}

圖片目標：
以具代表性的視覺設計呈現「{{成語}}」的概念。圖片必須讓觀者從畫面中的主體、場景、動勢、對比、象徵或情境關係理解「{{語義重點}}」，而不是依賴文字說明。

格式要求：
512x512，方形構圖。不得出現任何文字、標語、對話框、旁白、可讀招牌、題字、浮水印或印章。畫面需完整、清楚、可辨識，主題焦點明確。
若採用四格漫畫，請分成清楚的 2x2 格式，閱讀順序為左上、右上、左下、右下。每格角落可加入小型格次標號 1、2、3、4，標號僅作閱讀順序提示，不得加入其他文字、對話框或旁白。

視覺轉譯原則：
請優先轉譯成可被現代觀者理解的視覺情境，但不必限定於人物或日常生活。可依成語意境選擇人物互動、動物寓意、自然景觀、物件配置、空間關係、城市場景、職場情境、校園情境、運動情境、抽象象徵或超現實隱喻。典故只作概念來源，不應自動轉成古裝人物或古代場景。

視覺策略選擇：
請根據「{{關鍵概念}}」選擇最適合的呈現方式，而非固定使用人物敘事。
1. 若成語描述人格、關係、選擇、溝通或社會情境，可使用人物互動。
2. 若成語描述變化、趨勢、力量、困境、差距或抽象概念，可使用場景、物件、自然現象或象徵構圖。
3. 若成語本身有明確物件或動作，但字面畫面容易誤導，應改用能呈現語義的現代視覺隱喻。
4. 若成語需要因果或前後變化才能理解，可採單張畫面中的分區構圖、連續動勢或四格漫畫；若使用四格漫畫，需以 1、2、3、4 標示閱讀順序；若單一象徵畫面更有代表性，則不必使用四格。

畫面內容：
以「{{主要主體}}」為畫面核心，放置於「{{場景設定}}」中，透過「{{核心視覺關係}}」呈現「{{關鍵概念}}」。構圖採「{{構圖方式}}」。畫面應具體、可辨識、富有意境，避免只畫字面物件或單純符號堆疊。

視覺風格：
{{視覺風格}}。整體需有一致的美術語言、清楚的焦點層次與良好的光影或色彩控制。可以是插畫、四格漫畫、繪本風、編輯插畫、概念藝術、紙雕感、低多邊形、微縮模型、平面設計式構圖或其他合適風格，但不得使用寫實攝影感作為唯一表現手段，除非該成語特別適合。

必須避免：
1. 因為「成語」而自動出現古裝人物、古代建築、竹簡、毛筆、卷軸、朝堂、戰場、神獸、鳳凰、麒麟、泰山等典故字面元素，除非該成語語義不可避免。
2. 直接畫成字面解釋。
3. 呈現反義概念「{{反義排除}}」。
4. 使用文字、字幕、對話框或可讀符號來解釋成語。
5. 所有成語都套用同一種人物站位、職場情境或古風情境。
```
