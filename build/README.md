# Images 目錄檢查摘要

* * *

## 檔案

| 檔案 | 用途 |
| --- | --- |
| `idiom-image-meta-prompt-template.md` | 成語圖片提示詞範本 |
| `prompt_todo.md` | 追蹤每個主條成語是否已產生完整 Markdown 提示詞檔 |
| `image_todo.md` | 預備生成的成語圖片清單 |

* * *

## 數量摘要

| 清單 | 項目數 |
| --- | ---: |
| 完整圖片提示詞參數待辦清單 | 1664 |
| 預備生成的成語圖片清單 | 1664 |

* * *

## 輸出規則

- 圖片輸出目錄：`images/`
- 圖檔命名規則：`編號-1.png` 到 `編號-4.png`
- 清單內容：只保留主條成語
- `prompt_todo.md` 追蹤提示詞 Markdown 檔產出，`image_todo.md` 追蹤圖片檔案產出
- 每個成語一份提示詞檔：`images/prompts/編號.md`
