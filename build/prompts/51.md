# 51. 破鏡重圓

## 成語基本資料

| 欄位 | 內容 |
|---|---|
| 編號 | 51 |
| 成語 | 破鏡重圓 |
| 主條成語／非主條成語 | 主條成語 |
| 釋義 | 破成兩半的鏡子再次復合。指南朝陳徐德言與妻樂昌公主於戰亂離散時各執半鏡，作為他日相見的信物，後果因此得以相聚。典出唐．孟棨《本事詩．情感》。後用「破鏡重圓」比喻夫妻離散或感情決裂後重新團圓合好 |
| 用法說明-語義說明 | 比喻夫妻離散或情感決裂後重新團圓合好。 |
| 用法說明-使用類別 | 用在「別離重聚」的表述上。 |
| 近義成語 | 言歸於好 |
| 反義成語 | 分釵破鏡、分釵斷帶、勞燕分飛、覆水難收、鸞飄鳳泊 |
