# 1027. 苟且偷安

## 成語基本資料

| 欄位 | 內容 |
|---|---|
| 編號 | 1027 |
| 成語 | 苟且偷安 |
| 主條成語／非主條成語 | 主條成語 |
| 釋義 | 「苟且」，行事馬虎草率，得過且過。語出《漢書．卷八六．何武王嘉師丹傳．王嘉》。「偷安」，貪圖眼前的安逸，不顧將來可能發生的危難。語出漢．賈誼《新書．數寧》。「苟且偷安」形容得過且過，只圖眼前安逸，不顧將來 |
| 用法說明-語義說明 | 形容得過且過，只圖眼前安逸，不顧將來。 |
| 用法說明-使用類別 | 用在「貪圖安逸」的表述上。 |
| 近義成語 | 因循苟且、苟且偷生、苟安旦夕、得過且過 |
| 反義成語 | 自強不息、發憤努力、發憤圖強、發奮圖強、奮發向上 |
