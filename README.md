# 《成語典》資料下載

**這個資料庫整理並保存教育部《成語典》公開下載資料，方便直接取用原始文字資料、YAML 版本、精簡版本與字圖檔。**

* * *

## 簡介

本資料庫收錄教育部語文成果入口網「教育部國語辭典公眾授權網」所提供的《成語典》下載資料。依官方頁面記載，這批資料的最新版本編號為 `2020_20260324`，主要適用對象為語文教學者與學生，內容包含成語條目、注音、漢語拼音、釋義、典源、典故說明、書證、用法說明、近反義成語、參考詞語，以及部分條目對應的字圖資料。

目前這個資料庫已保存下列內容：

- `dict_idioms_2020_20260324.json`：完整 JSON 資料。
- `dict_idioms_2020_20260324.yml`：完整 YAML 資料。
- `dict_idioms_2020_20260324.min.json`：精簡 JSON 資料。
- `dict_idioms_2020_20260324.min.yml`：精簡 YAML 資料。
- `downloads/`：官方原始下載檔與試算表備份。

依本資料庫現有檔案實際統計：

| 項目 | 數量 |
| --- | ---: |
| 成語條目 | 5,489 |
| 字圖檔案 | 3,973 |

* * *

## 資料特色

- 收錄內容完整，適合用於語文研究、教學素材整理、資料檢索與程式化處理。
- 提供 JSON 與 YAML 兩種常見格式，方便不同工具鏈或工作流程使用。
- 同時保留精簡版本與字圖資料，便於依需求選擇較輕量或較完整的資料集。
- 保留 `downloads/` 目錄中的原始下載檔，方便追溯來源與重新比對。

* * *

## 資料結構概覽

`dict_idioms_2020_20260324.json` 的頂層結構為一個物件，主要鍵名如下：

```json
{
  "idioms": [
    {
      "編號": 1,
      "成語": "一毛不拔",
      "注音": "……",
      "漢語拼音": "……",
      "釋義": ["……"],
      "典源文獻名稱": "……",
      "典源文獻內容": ["……"],
      "典源注解": ["……"],
      "典故說明": "……",
      "書證": ["……"],
      "用法說明-語義說明": "……",
      "用法說明-使用類別": "……",
      "用法說明-例句": ["……"],
      "近義成語": "……",
      "反義成語": "……",
      "參考詞語": "……",
      "主條成語／非主條成語": "主條成語"
    }
  ]
}
```

常見欄位包含：

- 基本資訊：`編號`、`成語`、`注音`、`漢語拼音`
- 義項內容：`釋義`
- 文獻脈絡：`典源文獻名稱`、`典源文獻內容`、`典源注解`、`典源參考資料`
- 教學說明：`典故說明`、`書證`
- 用法資訊：`用法說明-語義說明`、`用法說明-使用類別`、`用法說明-例句`
- 辨識資訊：`辨識-形音辨誤`、`辨識-同`、`辨識-異`、`辨識-例句`
- 延伸資訊：`近義成語`、`反義成語`、`參考詞語`

部分欄位為字串，部分欄位為字串陣列；實際使用前應先依欄位型別處理。

## 使用情境

這份資料適合用於下列用途：

- 建立成語檢索服務
- 製作語文教學材料
- 進行自然語言處理前處理
- 建立詞條索引、全文檢索或向量化實驗資料集
- 研究成語典故、語義分類與用法說明

**但官方來源授權明確禁止改作，使用前應先確認自己的重製、散布、傳輸與公開使用方式符合授權條件與官方使用說明。**
