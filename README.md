# 成語星球

成語星球是一個以《成語典》為基礎的成語學習與查詢入口，整理可直接啟用的資料與前端展示頁面，讓使用者能快速瀏覽、檢索與連結相關成語資訊。

* * *

## 專案簡介

本專案整合教育部公開釋出的《成語典》內容，提供可直接載入前端應用的資料格式與基本結構，適合作為語文教學、研究、資料視覺化與文字分析的基礎資源。資料同時保留成語條目、音韻資訊、釋義、文獻脈絡與用法說明，並保有原始下載資料的保存位址，方便回溯與比對。

本資料庫目前保存版本編號為 `2020_20260324`，包含以下主要內容：

- `dict_idioms_2020_20260324.json`：完整 JSON 資料。
- `dict_idioms_2020_20260324.yml`：完整 YAML 資料。
- `dict_idioms_2020_20260324.min.json`：精簡 JSON 資料。
- `dict_idioms_2020_20260324.min.yml`：精簡 YAML 資料。
- `downloads/`：官方原始下載檔與試算表備份。

## 資料結構總覽

`dict_idioms_2020_20260324.json` 的頂層結構為一個物件，主要鍵名如下：

```json
{
  "idioms": [
    {
      "編號": 1,
      "成語": "一毛不拔",
      "注音": "……",
      "漢語拼音": "……",
      "釋義": ["……"],
      "典源文獻名稱": "……",
      "典源文獻內容": ["……"],
      "典源注解": ["……"],
      "典故說明": "……",
      "書證": ["……"],
      "用法說明-語義說明": "……",
      "用法說明-使用類別": "……",
      "用法說明-例句": ["……"],
      "近義成語": "……",
      "反義成語": "……",
      "參考詞語": "……",
      "主條成語／非主條成語": "主條成語"
    }
  ]
}
```

常見欄位分類如下：

- 基本資料：`編號`、`成語`、`注音`、`漢語拼音`
- 義項與用法：`釋義`、`典故說明`、`書證`、`用法說明-語義說明`、`用法說明-使用類別`、`用法說明-例句`
- 辨識資料：`辨識-形音辨誤`、`辨識-同`、`辨識-異`、`辨識-例句`
- 延伸與關聯：`近義成語`、`反義成語`、`參考詞語`
- 文獻脈絡：`典源文獻名稱`、`典源文獻內容`、`典源注解`、`典源參考資料`

欄位型別混合為字串與陣列，實際使用前請依欄位型別處理。

## URL 查詢參數

網頁可以用 query string 傳入條件，支援參數如下：

- `q`：搜尋關鍵字字串，可直接輸入關鍵詞；支援中文逗號 `，` 或英文逗號 `,` 分隔多項關鍵字。
- `match`：成語名稱比對模式，值為：
  - `exact`：只比對成語全名是否完全相符（僅限 `成語` 欄位）。
  - `prefix`：比對成語名稱開頭。
  - `suffix`：比對成語名稱結尾。
  - `any`：預設值，不指定 `match` 時使用，行為採現行關鍵字邏輯（見下）。
- `filter`：結果類型過濾，值為 `all`、`main`、`phrase`、`story`、`quiz`，未提供時預設 `main`。
  - `all`：全部
  - `main`：主條成語
  - `phrase`：有參考詞語
  - `story`：有典故
  - `quiz`：有趣味測驗
- `id`：直接指定成語識別值，開啟對應成語明細。

`match=any` 時的 `q` 規則：
- 有 `,`、`，`、空白字元時，這三者皆作為分隔符，條件為 OR（任一項命中即可）。
- 舉例：`q=自我,努力 勇敢` 會視為三個條件。

`match` 若未提供、或值不合法時，會回退為 `any`。
